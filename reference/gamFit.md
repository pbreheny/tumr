# Generalized Additive Model for Tumor Growth Data

Generalized Additive Model for Tumor Growth Data

## Usage

``` r
gamFit(
  tumr_obj = NULL,
  formula = NULL,
  data = NULL,
  id = NULL,
  time = NULL,
  measure = NULL,
  group = NULL,
  n_grid = 20,
  ...
)
```

## Arguments

- tumr_obj:

  A `tumr` object created by
  [`tumr`](https://pbreheny.github.io/tumr/reference/tumr.md).

- formula:

  GAM formula. Defaults to
  `log(measure + 1) ~ group + s(time, by = group) + s(id, bs = "re") + s(id, time, bs = "re")`.

- data:

  A data frame of tumor growth measurements.

- id:

  Column name of subject IDs.

- time:

  Column name of repeated time measurements.

- measure:

  Column name of repeated tumor measurements.

- group:

  Column name of treatment group assignments.

- n_grid:

  Number of time grid points for prediction. Default `20`.

- ...:

  Further arguments passed to
  [`gam`](https://rdrr.io/pkg/mgcv/man/gam.html).

## Value

An object of class `"tumr_gam"` with components:

- `fit`:

  The [`mgcv::gam`](https://rdrr.io/pkg/mgcv/man/gam.html) object.

- `data`:

  Processed data with internal column names.

- `formula`:

  The formula used.

- `relevant_info`:

  Named list of column names and reference group.

- `n_grid`:

  Number of prediction grid points.
