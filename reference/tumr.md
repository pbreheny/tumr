# Create tumr object

Create tumr object

## Usage

``` r
tumr(data, id, time, measure, group)
```

## Arguments

- data:

  Data frame

- id:

  Column of subject ID's

- time:

  Column of repeated time measurements

- measure:

  Column of repeated measurements of tumor

- group:

  Column specifying the treatment group for each measurement

## Value

A tumr object
